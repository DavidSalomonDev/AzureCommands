---
title: Listar grupos por etiqueta
description: Filtra los grupos de recursos que tienen una etiqueta concreta.
shell: azurecli
category: Grupos de recursos
tags: [grupo, resource-group, listar, tag, etiqueta, filtrar, get]
template: "az group list --tag <tagName>=<tagValue> --output <output>"
parameters:
  - name: tagName
    label: Nombre de la etiqueta
    type: string
    required: true
    placeholder: entorno
  - name: tagValue
    label: Valor de la etiqueta
    type: string
    required: true
    placeholder: produccion
  - name: output
    label: Formato de salida
    type: enum
    required: false
    default: table
    options: [table, json, jsonc, tsv, yaml]
---

Útil para agrupar por entorno, proyecto o centro de coste. Solo devuelve grupos cuya etiqueta coincida en nombre **y** valor.
