---
title: Listar grupos de seguridad de red
description: Muestra los Network Security Groups (NSG) de un grupo de recursos.
shell: azurecli
category: Redes
tags: [red, nsg, seguridad, listar, get]
template: "az network nsg list --resource-group <resourceGroup> --output <output>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: output
    label: Formato de salida
    type: enum
    required: false
    default: table
    options: [table, json, jsonc, tsv, yaml]
---

Para ver las reglas de un NSG concreto: `az network nsg rule list --resource-group <rg> --nsg-name <nsg> --output table`.
