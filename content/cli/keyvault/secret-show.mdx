---
title: Leer un secreto de Key Vault
description: Muestra el valor de un secreto almacenado.
shell: azurecli
category: Key Vault
tags: [keyvault, secreto, consulta]
template: "az keyvault secret show --vault-name <vaultName> --name <secretName> --query value --output tsv"
parameters:
  - name: vaultName
    label: Nombre del Key Vault
    type: string
    required: true
    placeholder: mi-keyvault-01
  - name: secretName
    label: Nombre del secreto
    type: string
    required: true
    placeholder: db-password
---
`--query value --output tsv` devuelve solo el valor, ideal para asignarlo a una variable de entorno.
