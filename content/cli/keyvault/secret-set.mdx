---
title: Guardar un secreto en Key Vault
description: Crea o actualiza un secreto dentro de un Key Vault.
shell: azurecli
category: Key Vault
tags: [keyvault, secreto, set]
template: "az keyvault secret set --vault-name <vaultName> --name <secretName> --value <secretValue>"
parameters:
  - name: vaultName
    label: Nombre del Key Vault
    type: string
    required: true
    placeholder: mi-keyvault-01
  - name: secretName
    label: Nombre del secreto
    type: string
    required: true
    placeholder: db-password
  - name: secretValue
    label: Valor del secreto
    type: string
    required: true
    placeholder: (valor sensible)
---
⚠️ Evita pasar valores sensibles en texto plano; considera `--file` para leerlos de un archivo y no dejarlos en el historial del shell.
