---
title: Crear un endpoint de Front Door
description: Crea un endpoint público dentro de un perfil de Front Door.
shell: azurecli
category: Front Door
tags: [front-door, afd, endpoint, crear]
template: "az afd endpoint create --resource-group <resourceGroup> --profile-name <profileName> --endpoint-name <endpointName> --enabled-state <enabledState>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: profileName
    label: Nombre del perfil
    type: string
    required: true
    placeholder: mi-frontdoor
  - name: endpointName
    label: Nombre del endpoint
    type: string
    required: true
    placeholder: mi-endpoint
  - name: enabledState
    label: Estado
    type: enum
    required: false
    default: Enabled
    options: [Enabled, Disabled]
---

El nombre del endpoint genera un host único `<endpoint>-<hash>.z01.azurefd.net`. Luego asocia orígenes (`az afd origin-group create` + `az afd origin create`) y rutas (`az afd route create`).
