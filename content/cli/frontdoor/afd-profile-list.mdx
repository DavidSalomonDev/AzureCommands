---
title: Listar perfiles de Front Door
description: Muestra los perfiles de Azure Front Door de un grupo de recursos.
shell: azurecli
category: Front Door
tags: [front-door, afd, listar, consulta, get]
template: "az afd profile list --resource-group <resourceGroup> --output <output>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: output
    label: Formato de salida
    type: enum
    required: false
    default: table
    options: [table, json, jsonc, tsv, yaml]
---

Para ver los endpoints de un perfil usa `az afd endpoint list --resource-group <rg> --profile-name <perfil> --output table`.
