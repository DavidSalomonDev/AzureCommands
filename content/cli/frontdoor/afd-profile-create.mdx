---
title: Crear un perfil de Front Door
description: Crea un perfil de Azure Front Door Standard/Premium (CDN global + WAF).
shell: azurecli
category: Front Door
tags: [front-door, afd, cdn, red, crear]
template: "az afd profile create --resource-group <resourceGroup> --profile-name <profileName> --sku <sku>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: profileName
    label: Nombre del perfil
    type: string
    required: true
    placeholder: mi-frontdoor
  - name: sku
    label: SKU
    type: enum
    required: false
    default: Standard_AzureFrontDoor
    options: [Standard_AzureFrontDoor, Premium_AzureFrontDoor]
---

Front Door es un servicio **global**, no ligado a una región. El SKU `Premium` añade WAF administrado y Private Link a los orígenes. Tras el perfil, crea un endpoint con `az afd endpoint create`.
