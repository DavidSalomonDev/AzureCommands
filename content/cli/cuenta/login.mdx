---
title: Iniciar sesión en Azure
description: Autentica la CLI con tu cuenta de Azure abriendo el navegador.
shell: azurecli
category: Cuenta y sesión
tags: [login, autenticación]
template: "az login"
parameters: []
---
En Cloud Shell ya estás autenticado. Para entornos sin navegador usa `az login --use-device-code`. Para una entidad de servicio: `az login --service-principal -u <appId> -p <password> --tenant <tenant>`.
