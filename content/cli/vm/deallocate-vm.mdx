---
title: Detener y desasignar una VM
description: Detiene la VM y libera el cómputo para dejar de facturar.
shell: azurecli
category: Máquinas virtuales
tags: [vm, stop, deallocate, ahorro]
template: "az vm deallocate --resource-group <resourceGroup> --name <vmName>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: vmName
    label: Nombre de la VM
    type: string
    required: true
    placeholder: mi-vm
---
A diferencia de `az vm stop`, **deallocate** libera el hardware y detiene la facturación por cómputo (sigues pagando el disco).
