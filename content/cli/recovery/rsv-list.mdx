---
title: Listar almacenes de Recovery Services
description: Muestra los Recovery Services Vault de un grupo de recursos.
shell: azurecli
category: Recovery Services Vault
tags: [backup, recovery, vault, listar, consulta]
template: "az backup vault list --resource-group <resourceGroup> --output <output>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: output
    label: Formato de salida
    type: enum
    required: false
    default: table
    options: [table, json, jsonc, tsv, yaml]
---

Omite `--resource-group` para listar todos los almacenes de la suscripción.
