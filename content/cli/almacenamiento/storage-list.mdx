---
title: Listar cuentas de almacenamiento
description: Muestra las cuentas de almacenamiento de un grupo de recursos.
shell: azurecli
category: Almacenamiento
tags: [storage, listar]
template: "az storage account list --resource-group <resourceGroup> --output <output>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: false
    placeholder: mi-grupo-recursos
  - name: output
    label: Formato de salida
    type: enum
    required: false
    default: table
    options: [table, json, jsonc, tsv]
---
Omite `--resource-group` para listar las cuentas de toda la suscripción.
