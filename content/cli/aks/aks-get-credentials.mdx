---
title: Obtener credenciales de un clúster AKS
description: Descarga las credenciales y las combina en tu kubeconfig local.
shell: azurecli
category: AKS
tags: [aks, kubernetes, credenciales, kubectl]
template: "az aks get-credentials --resource-group <resourceGroup> --name <clusterName>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: clusterName
    label: Nombre del clúster
    type: string
    required: true
    placeholder: mi-cluster-aks
---
Después ya puedes usar `kubectl get nodes`. Usa `--admin` para credenciales de administrador (omite Azure AD).
