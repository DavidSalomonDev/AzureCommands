---
title: Crear una red virtual
description: Crea una VNet con una subred inicial.
shell: powershell
category: Redes
tags: [red, vnet, crear]
template: "New-AzVirtualNetwork -ResourceGroupName <resourceGroup> -Name <vnetName> -Location <location> -AddressPrefix <addressPrefix>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: vnetName
    label: Nombre de la VNet
    type: string
    required: true
    placeholder: mi-vnet
  - name: location
    label: Región
    type: enum
    required: true
    default: East US
    options: [East US, West US 2, West Europe, North Europe, Brazil South]
  - name: addressPrefix
    label: Espacio de direcciones
    type: string
    required: false
    default: 10.0.0.0/16
---
Para añadir subredes crea objetos con `New-AzVirtualNetworkSubnetConfig` y pásalos con `-Subnet`.
