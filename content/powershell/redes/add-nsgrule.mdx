---
title: Crear una regla de NSG
description: Añade una regla de seguridad de entrada a un Network Security Group.
shell: powershell
category: Redes
tags: [red, nsg, seguridad, regla, crear]
template: "Get-AzNetworkSecurityGroup -ResourceGroupName <resourceGroup> -Name <nsgName> | Add-AzNetworkSecurityRuleConfig -Name <ruleName> -Priority <priority> -Direction Inbound -Access <access> -Protocol <protocol> -SourceAddressPrefix * -SourcePortRange * -DestinationAddressPrefix * -DestinationPortRange <ports> | Set-AzNetworkSecurityGroup"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: nsgName
    label: Nombre del NSG
    type: string
    required: true
    placeholder: mi-nsg
  - name: ruleName
    label: Nombre de la regla
    type: string
    required: true
    placeholder: permitir-https
  - name: priority
    label: Prioridad
    type: number
    required: true
    default: "100"
    placeholder: "100"
  - name: ports
    label: Puerto(s) de destino
    type: string
    required: true
    placeholder: "443"
  - name: access
    label: Acción
    type: enum
    required: false
    default: Allow
    options: [Allow, Deny]
  - name: protocol
    label: Protocolo
    type: enum
    required: false
    default: Tcp
    options: [Tcp, Udp, "*"]
---

En PowerShell la regla se añade en tres pasos encadenados: obtener el NSG, agregar la configuración de regla y guardar (`Set-AzNetworkSecurityGroup`). La prioridad debe ser única (100-4096, menor = más prioritaria).
