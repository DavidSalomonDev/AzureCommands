---
title: Crear un Azure Container Registry
description: Crea un registro privado de imágenes de contenedor.
shell: powershell
category: Container Registry
tags: [acr, registry, contenedores, crear]
template: "New-AzContainerRegistry -ResourceGroupName <resourceGroup> -Name <registryName> -Sku <sku> -EnableAdminUser"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: registryName
    label: Nombre del registro
    type: string
    required: true
    placeholder: miregistro
  - name: sku
    label: SKU
    type: enum
    required: false
    default: Basic
    options: [Basic, Standard, Premium]
---

El nombre del registro debe ser único globalmente (solo letras y números). `-EnableAdminUser` habilita el usuario administrador para autenticación rápida; para producción prefiere identidades administradas o tokens.
