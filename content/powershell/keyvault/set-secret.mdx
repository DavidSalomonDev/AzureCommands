---
title: Guardar un secreto en Key Vault
description: Crea o actualiza un secreto dentro de un Key Vault.
shell: powershell
category: Key Vault
tags: [keyvault, secreto, set]
template: "Set-AzKeyVaultSecret -VaultName <vaultName> -Name <secretName> -SecretValue (ConvertTo-SecureString <secretValue> -AsPlainText -Force)"
parameters:
  - name: vaultName
    label: Nombre del Key Vault
    type: string
    required: true
    placeholder: mi-keyvault-01
  - name: secretName
    label: Nombre del secreto
    type: string
    required: true
    placeholder: db-password
  - name: secretValue
    label: Valor del secreto
    type: string
    required: true
    placeholder: (valor sensible)
---
⚠️ El valor debe convertirse a `SecureString`. Evita dejar secretos en texto plano en el historial de la consola.
