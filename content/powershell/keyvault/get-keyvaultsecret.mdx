---
title: Leer un secreto de Key Vault
description: Obtiene el valor de un secreto almacenado en un Key Vault.
shell: powershell
category: Key Vault
tags: [keyvault, secreto, leer, get]
template: "Get-AzKeyVaultSecret -VaultName <vaultName> -Name <secretName> -AsPlainText"
parameters:
  - name: vaultName
    label: Nombre del Key Vault
    type: string
    required: true
    placeholder: mi-keyvault
  - name: secretName
    label: Nombre del secreto
    type: string
    required: true
    placeholder: mi-secreto
---

`-AsPlainText` devuelve el valor como texto (requiere Az 3.4+). Sin ese parámetro obtienes un objeto `SecureString`. Lista los secretos con `Get-AzKeyVaultSecret -VaultName <vault>` (sin `-Name`).
