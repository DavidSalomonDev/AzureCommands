---
title: Importar credenciales de un clúster AKS
description: Descarga las credenciales del clúster a tu kubeconfig local.
shell: powershell
category: AKS
tags: [aks, kubernetes, credenciales, kubectl]
template: "Import-AzAksCredential -ResourceGroupName <resourceGroup> -Name <clusterName>"
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: clusterName
    label: Nombre del clúster
    type: string
    required: true
    placeholder: mi-cluster-aks
---
Añade `-Force` para sobrescribir credenciales existentes en el kubeconfig.
