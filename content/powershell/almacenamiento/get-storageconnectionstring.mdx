---
title: Obtener la cadena de conexión
description: Construye la connection string de una cuenta de almacenamiento a partir de su clave.
shell: powershell
category: Almacenamiento
tags: [storage, cuenta, connection-string]
template: "$key = (Get-AzStorageAccountKey -ResourceGroupName <resourceGroup> -Name <storageAccountName>)[0].Value; \"DefaultEndpointsProtocol=https;AccountName=<storageAccountName>;AccountKey=$key;EndpointSuffix=core.windows.net\""
parameters:
  - name: resourceGroup
    label: Grupo de recursos
    type: string
    required: true
    placeholder: mi-grupo-recursos
  - name: storageAccountName
    label: Nombre de la cuenta
    type: string
    required: true
    placeholder: micuentastorage
---

A diferencia de Azure CLI (`az storage account show-connection-string`), en PowerShell se compone la cadena tomando la primera clave de acceso. Útil para configurar `AZURE_STORAGE_CONNECTION_STRING`.
